1. **Decimal, Binary, Octal, Hexadecimal Systems**
- **Decimal (Base 10)**: Uses digits 0-9.
- **Binary (Base 2)**: Uses digits 0 and 1.
- **Octal (Base 8)**: Uses digits 0-7.
- **Hexadecimal (Base 16)**: Uses digits 0-9 and letters A-F.
2. **Binary Operations**
- **Addition and Subtraction** in binary form (carry and borrow methods).
3. **One’s and Two’s Complement Methods**
- **One's complement**: Involves flipping all bits.
- **Two's complement**: Involves flipping all bits and adding 1.
#### 2.2 Logic Function and Boolean Algebra
1. **Boolean Algebra**
- A mathematical system used for representing logical expressions with two values:
**True (1)** and **False (0)**.
2. **Boolean Functions and Truth Tables**
- **Truth Tables** represent the output for all possible input combinations.
3. **Logic Gates** - **AND, OR, NOT, NAND, NOR, XOR, XNOR** gates and their respective truth tables and symbols.
4. **Laws of Boolean Algebra** - Include **Commutative**, **Associative**, **Distributive**, **Identity**,
and **Complement Laws**. 5. **Verification of Boolean Laws**
- Using truth tables to prove Boolean laws.
