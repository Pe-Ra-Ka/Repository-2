1. **Definition, Characteristics, and Application of Computer**
- A **computer** is an electronic device capable of performing calculations, storing, retrieving, and processing data.
- **Characteristics**: Speed, Accuracy, Automation, Storage, Versatility, Diligence, and Connectivity.
- **Applications**: Education, Business, Healthcare, Government, Entertainment, and Communication.
2. **Evolution of Computer Technology**
- **First Generation (1940-1956)**: Vacuum tubes, large and expensive (e.g., ENIAC).
- **Second Generation (1956-1963)**: Transistors replaced vacuum tubes, smaller and more efficient.
- **Third Generation (1964-1971)**: Integrated Circuits (ICs) improved performance and size.
- **Fourth Generation (1971-Present)**: Microprocessors, personal computers, and the rise of the internet.
- **Fifth Generation (Future)**: Artificial Intelligence and Quantum Computing.
3. **Measurement Unit of Processing Speed and Storage Unit**
- **Processing Speed**: Measured in Hz (Hertz), KHz, MHz, and GHz.
- **Storage**: Byte (B), Kilobyte (KB), Megabyte (MB), Gigabyte (GB), Terabyte (TB).
4. **Supercomputers, Mainframe, Mini, and Microcomputers**
- **Supercomputers**: Extremely fast and powerful, used for complex simulations.
- **Mainframe Computers**: Large, powerful, used for bulk data processing.
- **Mini Computers**: Medium-sized, used by small organizations.
- **Microcomputers**: Personal computers (PCs), widely used in homes and offices.
5. **Mobile Computing and its Application** - Mobile computing refers to the ability to use computing devices wirelessly while moving.
It includes smartphones, laptops, tablets,and mobile apps for communication, entertainment, and productivity. #### 1.2 Computer System and I/O Devices 1. **Computer Architecture and Organization** - Describes how the components of a computer work together to perform tasks:
**Input Unit**, **Processing Unit**, **Memory Unit**, and **Output Unit**. 2. **Components of Computer System**
- **Input Unit**: Keyboard, mouse, scanner.
- **Output Unit**: Monitor, printer, speakers.
- **Processing Unit**: Central Processing Unit (CPU), where all processing occurs.
- **Memory Unit**: RAM (temporary), ROM (permanent storage), Cache (for faster access).
3. **Microprocessor**
- A microprocessor is the brain of the computer, responsible for processing data.
- **Clock Speed**: Determines how fast the microprocessor can process data (measured in GHz).
- **Word Length**: Refers to the number of bits the processor can handle at a time (e.g., 32-bit, 64-bit).
4. **Bus System**
- **Data Bus**: Carries data between components.
- **Address Bus**: Carries memory addresses.
- **Control Bus**: Carries control signals to coordinate actions.
5. **Primary Memory**
- **RAM**: Volatile, temporary memory.
- **ROM**: Non-volatile, permanent memory.
- **Cache**: Small, high-speed memory used to store frequently accessed data.
6. **Secondary Memory**
- **Magnetic Disk**: Hard drives and floppy disks.
- **Flash Memory**: SSDs, USB drives.
- **Optical Disk**: CDs, DVDs.
- **External Storage Devices**: External hard drives, cloud storage.
7. **Input Devices** - Devices such as **keyboard**, **mouse**, **scanner**, **light pen**,
**OMR**, **OCR**, **BCR**, **MICR**, **microphone**, and **digital cameras**
used to enter data into a computer. 8. **Output Devices** - Devices like **monitors** (LCD, LED), **printers** (Dot Matrix, Inkjet, Laser),
and **speakers** used to display or produce results.
9. **Hardware Interfaces** - **Ports**: **Parallel**, **Serial**, **USB**, **HDMI**.
- **Expansion Slots**: Used for adding additional components (e.g., sound cards, network cards).
