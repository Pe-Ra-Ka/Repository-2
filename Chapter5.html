#### 5.1 Programming Concept
1. **Introduction to Programming Languages**
- **Low-Level Languages**: Assembly, machine language.
- **High-Level Languages**: C, Python, Java.
- **4GL**: SQL, Visual Basic.
2. **Compiler, Interpreter, Assembler**
- **Compiler**: Translates entire programs at once.
- **Interpreter**: Translates programs line by line.
- **Assembler**: Converts assembly language to machine code.
3. **Control Structures**
- **Sequence**, **Selection**, and **Iteration**.
4. **Program Design Tools**
- **Algorithm**, **Flowchart**, and **Pseudocode**.
