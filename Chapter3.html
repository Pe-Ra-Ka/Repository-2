#### 3.1 Concept of Software
1. **Software Definition**
- Software refers to the programs or instructions that tell the computer how to perform tasks.
2. **Categories of Software**
- **System Software**: Operating systems, device drivers.
- **Utility Software**: Antivirus, disk management tools.
- **Application Software**: Word processors, spreadsheets.
- **Web-based Software**: Google Docs, Dropbox.
- **Mobile Apps**: Instagram, WhatsApp.
#### 3.2 Concept of Operating System
1. **Introduction to Operating System**
- Software that manages hardware resources and provides services to software applications.
2. **Role of Operating System**
- Manages resources, controls execution, handles input/output, and manages memory.
3. **Functions of an Operating System**
- Includes process management, memory management, file management, and security.
4. **Operating System Terminology**
- **Multiprogramming, Multitasking, Multiprocessing, Distributed Systems**.
#### 3.3 Windows Operating System
1. **GUI-Based Operating System**
- Provides a graphical user interface for user interaction.
- **Features**: Desktop, start menu, file explorer, control panel.
2. **Managing Files and Folders**
- Use **File Explorer** to navigate and manage files.
3. **Installing and Removing Devices**
- Use **Device Manager** for installing and removing hardware.
4. **Managing Privacy and Passwords**
- Set up user accounts and privacy settings.
#### 3.4 Open Sources and Mobile Operating System
1. **Open Source Operating Systems**
- Example: **Linux**, **UNIX**.
2. **Mobile Operating Systems**
- **iOS**, **Android**.
